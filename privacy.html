Політика конфіденційності

DIESEL-CRAFT обробляє контактні дані тільки для зв'язку з клієнтом, запису на сервіс, консультації та виконання робіт.

Які дані ми можемо отримувати

  • номер телефону, ім'я, повідомлення у формі або месенджері;
  • дані про авто, якщо клієнт передає їх для консультації;
  • технічні дані аналітики сайту: сторінка, джерело переходу, подія кліку.

Для чого використовуються дані

Для дзвінка, запису на діагностику або ремонт, уточнення адреси сервісу, підготовки попередньої консультації та покращення роботи сайту.

Передача третім особам

Ми не продаємо персональні дані. Дані можуть передаватися лише сервісам зв'язку, аналітики або карт, які потрібні для роботи сайту.
